# 上海理工大学短学期——招聘系统::前端

> Recruitment System Front

## 项目后端

[上海理工大学短学期——招聘系统::后端](https://github.com/what-is-me/Recruitment-System-Backend)

## 项目下载

```
git clone https://github.com/what-is-me/Recruitment-System-Front.git
cd Recruitment-System-Front
npm install
```

## 运行

```
npm run serve
```

## 使用技术框架

* vue3
* vue-router
* vuex
* naive ui
